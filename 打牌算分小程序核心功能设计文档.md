## 2. 积分计算逻辑设计

### 2.1 核心算分规则

#### 2.1.1 基础算分逻辑
- **谁输入谁扣分**：当前用户（输入分数的人）扣分
- **输入给谁谁加分**：被输入分数的玩家加分
- **分数传递**：输入的分数从当前用户转移到目标用户

#### 2.1.2 算分流程
1. 用户点击目标玩家的分数区域
2. 弹出分数输入框
3. 用户输入分数（正数）
4. 系统计算：
   - 当前用户分数 = 当前用户分数 - 输入分数
   - 目标用户分数 = 目标用户分数 + 输入分数
5. 生成系统消息，记录分数变动
6. 实时更新所有玩家的分数显示

### 2.2 分数类型

#### 2.2.1 基础分数
- **正数**：表示输分（从当前用户扣除，给目标用户）
- **不支持负数**：系统会自动处理为正数

#### 2.2.2 特殊情况处理
- **分数为0**：不进行任何操作
- **输入非数字**：提示用户输入有效数字
- **分数过大**：提示用户分数超出合理范围

## 5. 实现方案

### 5.1 积分计算实现

#### 5.1.1 前端实现
```javascript
// room.vue 中的分数处理函数
const handleScoreChange = (index) => {
  const score = parseInt(scoreInputs.value[index]);
  if (!isNaN(score) && score > 0) {
    // 谁输入谁扣分，输入给谁谁加分
    players.value[index].score += score;
    players.value[0].score -= score;
    // 添加实时字幕
    addSubtitle(`玩家 ${players.value[0].name} 输给 ${players.value[index].name} ${score} 分`);
    // 清空输入框
    scoreInputs.value[index] = '';
    // 取消编辑状态
    editingScoreIndex.value = -1;
  } else {
    // 取消编辑状态
    editingScoreIndex.value = -1;
  }
};
```

#### 5.1.2 后端实现
```javascript
// 云函数实现分数更新
export async function updateScores(roomId, fromUserId, toUserId, score) {
  // 更新数据库中的分数
  await db.collection('players').where({
    roomId,
    userId: fromUserId
  }).update({
    score: db.command.inc(-score)
  });
  
  await db.collection('players').where({
    roomId,
    userId: toUserId
  }).update({
    score: db.command.inc(score)
  });
  
  // 记录分数变动历史
  await db.collection('scoreHistory').add({
    roomId,
    fromUserId,
    toUserId,
    score,
    timestamp: new Date()
  });
}
```

// 云函数实现消息广播
export async function broadcastMessage(roomId, message) {
  // 保存消息到数据库
  await db.collection('messages').add({
    roomId,
    content: message.content,
    type: message.type,
    userId: message.userId,
    timestamp: new Date()
  });
  
  // 推送消息给房间内所有用户
  const players = await db.collection('players').where({ roomId }).get();
  players.forEach(player => {
    // 推送消息到用户
    pushMessage(player.userId, message);
  });
}
